README - Calicopia

A Califorina-focused produce app for Android

Development
-----------

Calicopia is the result of collaboration between USF's computer science 
deparment and art and architecture department forcused on civic engagement.

Its purpose is to aid comsumers in selecting fresh, local, seasonal produce.

Screenshots: imgur.com/a/quxlC

To download from Google Play: https://play.google.com/store/apps/details?id=com.dferreras.project#?t=W251bGwsMSwxLDIxMiwiY29tLmRmZXJyZXJhcy5wcm9qZWN0Il0.

For more notes on its development see: 
somerandomdude.com/2012/06/19/designing-civic-engagement-in-the-classroom-our-experiences

Installing
----------

Download Calicopia.apk. Can be run on any Andoroid phone or in an emulator.
CaliCopia requires Android 2.2 Froyo.
